# 日本を空色に (Sky Color Project) - プロトタイプ

「日本を空色に」プロジェクトの最小機能プロトタイプです。
ユーザーが選んだ色を日本地図上にプロットし、みんなの空の色を共有するコンセプトの実装です。

## ファイル構成

- `index.html`: メインのHTMLファイル。構造とSVG地図を含みます。
- `style.css`: スタイルシート。レスポンシブ対応と地図の装飾。
- `script.js`: アプリケーションロジック。localStorageへの保存、描画処理。

## セットアップ手順

特別なインストール手順はありません。ブラウザがあれば動作します。

1. このフォルダ内のファイルを全て同じ場所に保存してください。
2. `index.html` をダブルクリックして、お使いのWebブラウザ（Chrome, Edge, Safari, Firefoxなど）で開いてください。

## 動作確認手順

1. **初期表示確認**:
   - 画面に日本地図が表示されていることを確認してください。
   - 「今日の色を選ぶ」カラーピッカーが表示されていることを確認してください。

2. **投稿テスト**:
   - 地図上の任意の場所（陸地でも海でも可）をクリックしてください。赤い丸枠が表示されます。
   - 「今日の色を選ぶ」で好きな色に変更してください。
   - 「この場所に色を置く」ボタンをクリックしてください。
   - クリックした場所に、選んだ色の半透明な円が描画されることを確認してください。

3. **データ保存確認**:
   - ページをリロード（再読み込み）してください。
   - 先ほど投稿した円が消えずに表示されていることを確認してください。

4. **1日1回制限の確認**:
   - ページをリロードした後、再度地図をクリックしてみてください。
   - 「本日は既に投稿済みです」というメッセージが表示され、投稿ボタンが無効化されていることを確認してください。
   - ※テスト用に「データをリセット」ボタンを用意しています。これを押すとlocalStorageがクリアされ、再度投稿できるようになります。

5. **ツールチップ確認**:
   - 地図上の円にマウスカーソルを合わせてください。
   - 投稿日時と色コードが表示されることを確認してください。

## 技術メモ

- **データ保存**: ブラウザの `localStorage` を使用しています。サーバーは使用していません。
- **地図**: SVGを使用しており、座標計算はSVGの `viewBox` 座標系に基づいて行われます。画面サイズが変わっても位置ズレは起きません。
